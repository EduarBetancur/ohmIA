# Publicar imagenes con GitHub Pages

El repositorio remoto es:

```text
https://github.com/EduarBetancur/ohmIA
```

## Activar Pages

1. Entrar al repositorio en GitHub.
2. Ir a `Settings`.
3. Ir a `Pages`.
4. En `Build and deployment`, seleccionar:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guardar.
6. Esperar a que GitHub publique el sitio.

## URL del sitio

Cuando Pages este activo, la portada sera:

```text
https://eduarbetancur.github.io/ohmIA/
```

Estas URLs se pueden usar como enlaces en respuestas de texto o como imagenes si la plataforma permite respuestas enriquecidas.

Ejemplo:

```text
Mira este diagrama de LCK:
https://eduarbetancur.github.io/ohmIA/assets/nodo-lck.svg
```
